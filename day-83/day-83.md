#  Tempus
Welcome to **day 83** of 365 days of code - coding every day for a year, little and often

![Tempus Logo](./tempuslogofull.svg)

It's official, the timetable app is now called "Tempus". I really liked this name when it was suggested to me, it's latin for time (ooo latin, fancy), and I think it fits well with the feel of the app.

Anyway, one of the big parts was actually updating the README for the app so that it is relevant, not just the nextjs boilerplate code. That included thinking about how other people might run it that aren't me, and in particular from a self hosting perspective, making it easy. I know I need to do something about making the docker image available so that users don't have to build it themselves, and that's on the to-do list as well, but today feels like a pretty big milestone so I'm just going to enjoy that for now.

Anyway, no rest for the wicked, more tomorrow!
