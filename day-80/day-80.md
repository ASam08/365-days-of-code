#  Type Type Type
Welcome to **day 80** of 365 days of code - coding every day for a year, little and often

Today was definitely a lesson in typescript, and more importantly typing. I was bashing my head against a bit of a wall with the overload issues that VSC kept populating for the add timetable block component. Eventually I got there, being super thorough with the typing for the component, but it's a lesson learned, and something I'm going to have to spend a few days sorting out through the rest of the codebase.

Because this has been a learn on the go project from the beginning, I have some serious debt to go back and sort through with regards to my typing. Weirdly VSC is picking up these overload issues all of a sudden where it wasn't a few days ago, so I guess something has changed a little. ESLint is not picking up the overloads, but is picking up a ton of places where I have explicit any types, so... I also need to go back and fix those.

All part of the fun and the experience, but most impoprtantly, the learning. Given the knowledge I have now, I certainly would like to hope that I would have been doing this as I go, and I guess that's a habit I will have to form. A few days sorting through all of this will probably go a good way towards embedding that lesson...
