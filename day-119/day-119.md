#  Better-Auth - Day 2
Welcome to **day 119** of 365 days of code - coding every day for a year, little and often

Another productive day today. I migrated both the login form and signup form to client side auth, which took a little getting my head around, and a bit of tweaking of the DB. It also resulted in the removal of the account enabled and password fields from the users DB table, so another migration there.

I'm actually feeling pretty impressed by Better-Auth, it's feeling a little bit like the Drizzle experience, now that I'm getting my head around it, and it unlocks alot of extra functionality that would have been a pain to implement myself, such as email verification, password resets, password changes, account deletion, adding an admin portal for user management, etc. etc.

One thing I do need to do, I and I guess it should be a tomorrow job, is update my jest tests (*sigh*). I'll look forward to that...

More tomorrow!
