#  More Updates
Welcome to **day 166** of 365 days of code - coding every day for a year, little and often

A bit more of a boring day today. I started with some much needed package updates, ran the tests etc. all working, nice and easy.

I then took on the migration of pnpm from v10 to v11. A little bit of doco reading, thanks to the devs for the codemod, and away we went.

Finally I did some digging to work out why I had such an old version of esbuild in use, which was causing a bunch of vulnerabilities to be reported. It turns out that drizzle-kit had moved away from esbuild, but hadn't removed the dependancies. After looking into it, I decided the best way to go was to override it in pnpm, which is slightly different in v11 (good job I read that doco earlier).

Anyway, all of that is done, things are tidier and updated, and a new release out to celebrate.

More tomorrow!
