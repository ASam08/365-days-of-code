#  Definitely Not Email - Part 1
Welcome to **day 154** of 365 days of code - coding every day for a year, little and often

Ok, we're past the point of pretending now that the work today was about email, it definitely wasn't. Today was spent working out why the popover wasn't working on my mobile. After a bunch of going around in circles, I spotted something in the server startup logs. It turns out that I didn't have my local instance set up as an allowed origin, so when I tried to access it from my phone for testing, it was creating a CORS issue and therefore not interacting properly...Bit of an own goal really...

Anyway, I sorted that out, and actually the debugging did lead me to tidy up my master layout component, so I guess that's something...

Anyway, more tomorrow!
