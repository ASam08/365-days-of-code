#  Workaround No More
Welcome to **day 145** of 365 days of code - coding every day for a year, little and often

Amazingly, I managed to get everything done today, and so the workaround is a workaround no more. AUTH_ON is gone and now Auth is the requirement for all.

I ended up making some changes to the env variables check that I put into the startup component, moving TEMPUS_URL to an optional list, as it has a fallback (localhost:3000) so now that just shows up in the logs if you go looking as a good heads up.

And that's it! It has given me a few things to think about, and some other bits that I want to try out, I think the next one will be adding email integration so that we can do things like password resets and email validation, particularly handy now that we are forcing auth upon everyone.

I guess that means more tomorrow!
